## Run Locally

Clone the project

```bash
  git clone https://github.com/kennyoyaf/todo-list-mernstack.git
```
Go to the project directory

```bash
cd backend
```

Install dependencies

```bash
  npm install
```

Start the server

```bash
  npm start
```
